BUUCTF_WEB_Dropbox

2019-07-24

注册。

本来以为是普通的上传绕过，上传之后发现对文件只有 下载 和 删除 两个按钮。先上传一个普通文件，点击 下载 发现，post内容为filename=XXX.jpg。

使用BurpSuite的Repeater模块，post内容:

filename=file:///etc/passwd

回显passwd文件内容。

读一下源码，filename=file:///var/www/html/index.php(upload.php)(class.php)(download.php)(delete.php)

关键就在class.php的源码。

长代码预警。

emmmmmmmm phar反序列化。
给出大佬的脚本。


大佬WP地址

要改一下本地的php.ini文件配置。

php --ini得到配置文件地址。

vim /etc/php/7.3/cli/php.ini

phar.readonly = Off

OK.

运行一下大师傅的脚本，得到phar.phar文件，上传之(别忘抓包该一下文件类型为image/jpeg)

然后在delete.php页面，post:

filename=phar://phar.jpg/test
