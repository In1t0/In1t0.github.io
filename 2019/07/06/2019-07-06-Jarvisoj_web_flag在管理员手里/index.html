Jarvisoj_web_flag在管理员手里

2019-07-06

hash拓展攻击

只有管理员才能获得flag，你能想办法获得吗？

说到admin就想到了cookie,打开Cookie Editor，发现有role和hsh。

hsh: 3a4727d57463f122833d9e732f94e4e0

role: s%3A5%3A%22guest%22%3B //url解密得 s:5:”guest”; 又是php序列化。

扫一下隐藏信息。

[200] => index.php

[200] => index.php

wget下载index.php,重命名为.index.php.swp 。

vim -r index.php查看源码。

在role=guest的情况下，hsh在md5加密前应为"tseug":5:s

安装Hashpump

\x转换为%
;转换为%3b
CookieEditor
更改cookie值
